## Provider Audit

Provider-backed proposal packets now include `providerAudit` in:

- `proposal-status.json`
- `metrics.json`
- `provider-safety-report.json`

The audit object records backend, command hash, started/finished timestamps, duration, input/output/patch bytes, accepted/rejected flags, rejection reason, and null token/cost placeholders.
