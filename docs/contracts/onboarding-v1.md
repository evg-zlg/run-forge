# Onboarding JSON v1

`runforge onboarding --format json` is the stable discovery document described by `schemas/onboarding-v1.schema.json`. With `--repo` and optional `--working-directory`, it reports repository and execution roots separately and discovers package/validation metadata from the execution root. It contains product/version/transport identity, workflow, supported and unsupported interfaces, exact command templates, contract locations, primary artifacts, safety defaults, optional target inspection, project-file state, and one next action. It never emits environment variables, credentials, or tokens.
